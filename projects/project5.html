Project Name: Landing Page for Website Resellers

About: A landing page which intends on getting users to purchase their favourite template. The overall process consists of (1) writing a domain name, (2) choosing a template (after looking through a variety of different themes), (3) proceeding to select a pricing plan, and (4) paying through paypal.

Code Technologies: JavaScript ES6, HTML5, CSS3, Bootstrap 4, and Git for version control
